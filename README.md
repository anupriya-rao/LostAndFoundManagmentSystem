# 🎒 Lost and Found Item Management System

## 📌 Description

This is a **Java console-based application** designed to manage lost and found items within a university environment. The system allows users to report lost or found items, search items, and request claims, while admins manage approvals and user accounts.

---

## 🚀 Features

### 👤 User Features

* Register and login
* Report lost items
* Report found items
* View lost and found items
* Search items by name, category, or location
* Request to claim an item
* View claim request status

### 🛠️ Admin Features

* View all reported items
* Approve or reject claim requests
* Mark items as returned
* Delete invalid/fake reports
* Manage users

## ▶️ How to Run

### ✅ Prerequisites

* Java JDK 8 or higher
* IntelliJ IDEA / VS Code / any Java IDE

### 🔧 Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/lensen-degife/LostAndFoundManagmentSystem.git
   ```

2. Navigate to the project folder:

   ```bash
   cd LostAndFoundManagmentSystem
   ```

3. Open in your IDE

4. Run the program:

   * Open `Main.java`
   * Click **Run**
